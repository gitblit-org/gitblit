Unix/Linux/Mac OS X

1. This script should work out-of-the-box, assuming you have Python and Git.
2. Add the pt script to your path (perhaps move to ~/bin)


Windows

1. Download and install Python 2 or 3, if you have not (http://www.python.org)
2. Download and install Git for Windows, if you have not (http://git-scm.com)
3. Put the pt.cmd and pt.py file together in your path


Usage

    pt fetch <id> [-p,--patchset <n>]
    pt checkout <id> [-p,--patchset <n>] [--force]
    pt pull <id> [-p,--patchset <n>]
    pt push [<id>] [--force] [-m,--milestone <milestone>] [-t,--topic <topic>] [-cc <user> <user>]
    pt start <topic>
    pt propose [new | <branch> | <id>] [-m,--milestone <milestone>] [-t,--topic <topic>] [-cc <user> <user>]
    pt cleanup <id>
