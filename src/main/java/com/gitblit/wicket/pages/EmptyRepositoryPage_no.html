Tomt repository

[repository] er et tomt repository som ikke kan vises av Gitblit.

Vennligs push noen endringer inn til

Etter at du har pushe noen endringer, kan du oppfriske denne siden for \u00e5 se repositoriet.

Git kommando-linje syntaks

Hvis du ikke har et lokalt repository, b\u00F8r du klone dette repositoriet, committe noen filer og s\u00e5 skyve endringene tilbake til Gitblit. 


		

		Hvis du allerede har et lokalt repository med endringer i, kan legge dette repositoryet til som en remote, og pushe endringer til det.
		

		

		

		Hvis ditt repository er ment \u00e5 holdes i synk med et upstream repository, kan du legge det til.
